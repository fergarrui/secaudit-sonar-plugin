Untrusted HttpResponse redirect

This rule logs audit points when finding untrusted calls to HttpResponse.redirect. Make sure the parameter is validated, specially if it can come from the user.

Example: 

    class C extends HttpServlet {
        protected void doGet(HttpServletRequest request, HttpServletResponse response, String urlParam) throws ServletException, IOException {
            String url = request.getParameter("url");
            response.sendRedirect(url); // Dangerous redirect
        }
    }
