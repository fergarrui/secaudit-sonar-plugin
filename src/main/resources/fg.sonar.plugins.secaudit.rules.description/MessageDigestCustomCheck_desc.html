Message Digest Custom

This rule logs audit points when a custom MessageDigest is found. It is better to use recommended hashing algorithms: SHA-224, SHA-256, SHA-384, SHA-512, SHA-512/224, or SHA-512/256.

Example: 

class CustomMessageDigest extends MessageDigest { // Potentialy dangerous

    public CustomMessageDigest() {super("AlgorithmName");}

    @Override
    protected void engineUpdate(byte input) {
    }

    @Override
    protected void engineUpdate(byte[] input, int offset, int len) {
    }

    @Override
    protected byte[] engineDigest() {
    return new byte[0];
    }

    @Override
    protected void engineReset() {
    }
}
