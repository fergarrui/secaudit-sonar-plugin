Use of System.exit()

This rule logs audit points when usages of java.lang.System#exit() or java.lang.Runtime#exit() are found. If those methods are reachable in, for example, a web application, DOS attacks can be done, since it will shutdown its container.

Example: 

class A {
    void method() {
        System.exit(1); // Raises an audit point
        Runtime.getRuntime().exit(1); // Raises an audit point
    }
}

  • OWASP Denial of Service.
