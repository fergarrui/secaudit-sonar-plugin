File upload

This rule logs audit points when finding a file upload. Read 'see' section.

Frameworks detected:

  • Java Servlet - javax.servlet.http.Part
  • Apache commons - org.apache.commons.fileupload.FileItem
  • Spring - org.springframework.web.multipart.MultipartFile
  • Jersey - com.sun.jersey.core.header.FormDataContentDisposition

    • Example: 

    class TestServlet extends HttpServlet {
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws IOException, ServletException {
        Part filePart = req.getPart("file"); // An uploaded file is going to be processed
    }
}
