Unsafe reflection

This rule logs audit points when detects unsafe reflection. The method java.lang.Class#forName can receive input from an untrusted source and instantiate objects from improper classes. This can lead to unexpected behaviour, authentication bypass, control checks, etc.

Example: 

void method(String className) {
    Class a = Class.forName(className); // This input may be an user input.
    Object obj = a.newInstance();
}

See

  • CWE-470 - Use of Externally-Controlled Input to Select Classes or Code ('Unsafe Reflection').
  • OWASP Unsafe use of Reflection.
