HttpServlet present

This rule logs audit points when finding HttpServlet classes. In this class, there can be methods receiving requests. The parameters are accessed from HttpServletRequest. For example: HttpServletRequest.getParameter("anyParameter"). Some issues must be checked:

  • HTTP parameters
  • Authentication and authorisation
  • DB access
  • Command execution
  • File opening

Example: 

class A extends HttpServlet { // HttpServlet class found
  @Override
  public void doGet(HttpServletRequest request, HttpServletResponse response)
          throws IOException, ServletException {
    // This method processed a GET request.
  }
}
