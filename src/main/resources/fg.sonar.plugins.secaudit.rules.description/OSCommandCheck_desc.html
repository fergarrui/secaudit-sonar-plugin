OS Command Execution check

This rule logs audit points on potential OS command executions and the executions themselves.

Detects: 

    java.lang.Runtime.exec
    java.lang.ProcessBuilder
    java.lang.ProcessBuilder.command
    org.apache.commons.exec.Executor
    org.apache.commons.exec.Executor.execute

There is an OSCommandInjection rule for SonarQube : squid:S2076.

See

  • CWE-78 - Improper Neutralization of Special Elements used in an OS Command.
  • CWE-88 - Argument Injection or Modification.
