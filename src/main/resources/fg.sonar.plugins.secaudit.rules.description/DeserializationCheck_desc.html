Object deserialization

This rule logs audit points when finding deserialization. If the data is untrusted, can lead to deny of service, arbritary code execution, changes in application logic, etc.

Detects:

  • java.io.ObjectInputStream#readObject()
  • java.io.ObjectInputStream#readUnshared()

    • Example: 

    [...]
void method(FileInputStream fileInputStream) throws IOException, ClassNotFoundException {
    ObjectInputStream ois = new ObjectInputStream(fileInputStream);
    Object o = ois.readObject(); // deserialization
}
[...]

    See

    • CWE-502 - Deserialization of Untrusted Data.
    • OWASP Deserialization
    • ysoserial GitHub
    • FoxGlove Security post about exploiting deserialization.
