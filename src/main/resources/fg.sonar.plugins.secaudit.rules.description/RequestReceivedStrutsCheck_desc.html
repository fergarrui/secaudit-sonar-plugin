Struts1 request found.

This rule logs audit points when finding Struts1 requests. ActionForm form contains the request parameters. Some issues can be checked:

  • HTTP parameters
  • Authentication and authorisation
  • DB access
  • Command execution
  • File opening

Example: 

class A extends Action { // Request class

  public ActionForward execute(ActionMapping mapping, ActionForm form,
                               HttpServletRequest request, HttpServletResponse response) { // Method processing the request
  }
}
