Spring MVC Controller found

This rule logs audit points when finding Spring MVC controllers. All methods annotated with @RequestParam are receiving requests. Some issues can be checked:

  • HTTP parameters
  • Authentication and authorisation
  • DB access
  • Command execution
  • File opening

Example: 

@Controller
@RequestMapping("/")
class A { // This class is a Spring controller

  public String hello() {
    return "";
  }

  @RequestMapping("/anypath")
  public String anyPath() {
    return "";
  }
}
