File path traversal check

This rule logs audit points when a file is opened to read/write. The input parameters must be checked. If they are coming from a malicious user, this issue may lead to a Path traversal vulnerability.

Detects:

  • java.io.File
  • java.io.FileInputStream
  • java.io.FileReader
  • java.io.RandomAccessFile
  • java.io.FileWriter
  • java.io.FileOutputStream

    • Example: 

    class A {
  void method(String fileName) throws IOException {
    File file1 = new File(fileName); // Untrusted input
    File file2 = new File("file.txt");
  }
}

    See

    • CWE-22 - Improper Limitation of a Pathname to a Restricted Directory ('Path Traversal')
    • OWASP Path Traversal
