JAX-RS Rest web service

This rule logs audit points when finding JAX-RS Rest web services. In this class, there can be several methods receiving requests. Some issues must be checked:

  • HTTP parameters
  • Authentication and authorisation
  • Investigate CSRF

Example: 

@Path("path")
class RestController { // JAX-RS WS class found

  @GET
  String getEverything() {
    return "{}";
  }

  @GET @Path("{param}")
  String getSomething(@PathParam("param") String param) {
    return "{}";
  }
}
